RESUMEN
▪El núvol i la virtualització fan que actualment moltes de les operacions que realitzem s'emmagatzemin en
servidors en Internet.
▪ Els serveis del núvol són escalables i permeten oferir als usuaris la possibilitat de pagar només pels recursos
consumits.
▪ El núvol ofereix diferents tipus de serveis i diferents models en funció tant de les prestacions com del seu àmbit
d'aplicació.
▪ Existeixen múltiples aplicacions en el núvol que permeten realitzar magatzematge d'arxius, compartir
informació, realitzar control de versions o crear documents.
▪ Les aplicacions d'ofimàtica web són molt utilitzades i ofereixen grans prestacions; no solament les típiques de
processament, càlcul o presentacions, sinó que és possible afegir altres noves. Aquestes aplicacions ofereixen
moltes aplicacions d'ofimàtica web les funcionalitats de la qual són cada vegada més similars a les d'escriptori.
▪ Una característica comuna a totes les aplicacions web és la possibilitat de fer treball col·laboratius.
▪ Existeix la possibilitat d'integrar les aplicacions web creant accessos directes que emmarquen els programes a
través d'una finestra similar a la d'un programa d'escriptori.
▪ A més d'existir aplicacions d'ofimàtica web, hi ha sistemes operatius web que ofereixen els anomenats
escriptoris virtuals.
▪ Les aplicacions *SSB permeten integrar les aplicacions web de forma centralitzada perquè es comportin com a
programes d'escriptori.
