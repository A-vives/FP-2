Software propietario
Software libre - Este permite a millones de programadores aplicar y ampliar el crecimiento.

    Copyleft - Tiene restricciones que son de obligatorio cumplimiento

    GPL - Libre y de codigo abierto, no tiene porque ser gratuito , se puede altrar, copiar y distribuir de forma gratuita
    Debian - Exige que todo el software distribuido sea libre y vaya acompañado del codigo fuente
    BSD - Permite comercializar el software de forma libre, sin obligación de compartir el codigo fuente.
    MPL - Nos permite modificar, pero hay que mandarle una copia de las modificaciones perdiondo los derechos de dicha modificaciones
otros tipos de licencias
    Shareware - Utilizacion limitada en tiempo del software : periodo de prueba.
    Freeware - NO se puede modificar el codigo pero si usarlo de forma gratüita si utilizarlo de forma gratüita
    Privates - Codigo cerrado, nunca es gratis, no puede ser alterado ni copiado, software pre-instalado
